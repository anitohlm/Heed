# Technical Doc Screenshots

The Word doc generated by `generate_azure_doc.py` references screenshots
from this folder. If a file is missing the doc still builds — a styled
placeholder appears in its place — but for the polished version drop the
PNGs in here using the exact filenames below, then re-run:

```bash
python generate_azure_doc.py
```

## Filename conventions

- Lowercase, hyphen-separated.
- Prefix with the section number (`10-…`) so future sections can be added without renames.
- PNG format, full screenshot or relevant region — the generator scales to 6 inches wide by default.
